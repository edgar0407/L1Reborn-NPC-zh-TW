Shortcut Key Window Use:
Items and magic icons can be assigned to 8 shortcut key slot which is above the icon bar in right lower screen.

To assign, open the item window or magic window, and drag an item or magic icon while pressing left mouse button over to the corresponding slot and release the button.

Each slot is linked to F5~F12 keys and each key can be used by pressing.

For a Wheel Mouse, by rolling the wheel move the focus and then press the wheel to use.

Or each slot can be double left clicked on the mouse.

The sets that have saved shortcut keys can be changed or additionally set up to another shortcut key set by using F1~F3keys.

The shortcut key sets can be setup in many ways for a total of 24 magic or items.
