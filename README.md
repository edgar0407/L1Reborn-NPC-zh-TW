# 天堂 Reborn NPC 對話檔繁體中文化

天堂（Lineage）Reborn 私服 NPC 對話檔的繁體中文翻譯補完專案。

---

## 專案背景

Reborn 伺服器的遊玩環境以英文為主（`-e.html`），  
原本的繁中對話檔（`-c.html`）存在大量亂碼（ANSI 轉 UTF-8 問題）。  
本專案目標是將所有 NPC 對話補完為可正常顯示的繁體中文版本。

---

## 檔案說明

| 副檔名命名 | 語系 |
|-----------|------|
| `npcname-c.html` | 繁體中文（本專案產出） |
| `npcname-e.html` | 英文（Reborn 伺服器原版） |
| `npcname-j.html` | 日文 |
| `npcname-k.html` | 韓文 |

- 所有 HTML 檔案使用 **ANSI 編碼**，請勿以 UTF-8 儲存
- 對話語法為類 HTML 格式，顏色代碼非標準 HTML 色碼

---

## 如何使用

1. 下載本倉庫的所有 `-c.html` 檔案
2. 透過伺服器工具（pack 打包）覆蓋進遊戲資料夾
3. 啟動 Reborn 客戶端即可看到繁體中文 NPC 對話

---

## 翻譯原則

1. **Reborn `-e.html` 為最終權威**：伺服器跑 Reborn，內容以 Reborn 為準
2. **Real 繁中版可直接套用的條件**：
   - Real 與 Reborn 的 link / action / src / href 完全一致
   - 內容語義相符（同一劇情版本）
3. **Real 與 Reborn 內容不同時**：以 Reborn 英文內容翻譯，不套用 Real

## 貢獻方式

歡迎協助人工複審 AI 翻譯的對話檔。  
由 AI 從英文翻譯的檔案，開頭會有以下標記：

```html
<!-- 烏薩奇: AI英文中 -->
```

標記代表該檔案尚待人工複審，歡迎提交修正。
