# 天堂 Reborn NPC 對話繁中化 (v1.0)

> **使用者文件**：NPC 對話檔下載與使用說明

天堂（Lineage）Reborn 私服 NPC 對話檔的繁體中文翻譯補完專案。  
修正原版 `-c.html` 大量亂碼問題，讓 Reborn 伺服器的 NPC 對話正確顯示繁體中文。

---

## 快速開始

### 1. 下載檔案

從本倉庫下載所有 `*-c.html` 檔案，或直接 Clone：

```bash
git clone https://github.com/edgar0407/L1Reborn-NPC-zh-TW.git
```

### 2. 套用至遊戲

將下載的 HTML 檔案透過伺服器打包工具（pack）覆蓋進遊戲的 NPC 對話資料夾，  
重新啟動伺服器後即可看到繁體中文 NPC 對話。

---

## 專案背景

Reborn 伺服器的遊玩環境以英文為主（`-e.html`），  
原本的繁中對話檔（`-c.html`）存在大量亂碼，推估為 ANSI 意外轉成 UTF-8 的問題。

本專案的修正流程：
1. 以 Reborn `-e.html`（英文版）為最終內容權威
2. 若 Real 繁中客戶端的 `-c.html` 連結結構與 Reborn 相符，直接套用
3. 結構不符時，從英文版翻譯並標記 `<!-- 烏薩奇: AI英文中 -->` 供人工複審

## 檔案說明

| 副檔名 | 語系 | 說明 |
|--------|------|------|
| `npcname-c.html` | 繁體中文 | 本專案產出，修正版本 |
| `npcname-e.html` | 英文 | Reborn 伺服器原版（內容權威） |
| `npcname-j.html` | 日文 | 參考用 |
| `npcname-k.html` | 韓文 | 參考用 |

> ⚠️ 所有 HTML 檔案使用 **ANSI 編碼**，請勿以 UTF-8 重新儲存，否則會產生亂碼。

## 貢獻方式

歡迎協助人工複審 AI 翻譯的對話檔。  
由 AI 從英文翻譯的檔案，開頭會有以下標記：

```html
<!-- 烏薩奇: AI英文中 -->
```

標記代表該檔案尚待人工複審，歡迎提交修正的 Pull Request。
