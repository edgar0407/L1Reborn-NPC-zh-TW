一級魔法：九級
雷霆風暴：

類型：攻擊、風

相性：中立

魔力／體力：48/0

期間：瞬間

目標：非玩家操縱的角色

範圍：施法者本身周圍3格

材料：無

效果：從大自然裡呼喚閃電及風的能量，所形成的魔法，能對被施法者及周圍四個單元內的敵人造成損害。

沉睡之霧：

類型：詛咒、水

相性：邪惡（每次使用增加10點邪惡值）

魔力／體力：40 / 0

期間：32秒

目標：其他玩家或非玩家操縱的角色

範圍：10格

材料：魔法寶石x1

效果：使目標三個單元內的人或怪物進入睡眠，一旦遭到攻擊便會醒來。

聖結界：

類型：轉換

相性：正義

魔力／體力：30 / 0

期間：32秒

目標：施法者本身及血盟成員

範圍：全畫面

材料：魔法寶石x2

效果：提高目標對各種傷害的抵抗力，施放他人減免的傷害將依照施法者的等級決定，施放自己則是傷害減半。

集體傳送術：

類型：控制

相性：中立

魔力／體力：60 / 0

期間：瞬間

目標：施法者本身及血盟成員

範圍：2格

材料：魔法寶石x1

效果：能將施法者二個單元內的血盟成員一同傳送到任意的位置，若搭配『傳送控制戒指』就能指定傳送的位置。

火風暴：

類型：攻擊、火、風

相性：中立

魔力／體力：48 / 0

期間：瞬間

目標：非玩家操縱的角色

範圍：施法者本身周圍3格

材料：無

效果：在施法者周圍引發具有火屬性的暴風。

藥水霜化術：

類型：操作

相性：中立

魔力／體力：20 / 0

期間：16秒

目標：其他玩家或非玩家操縱的角色

範圍：3格

材料：魔法寶石x2

效果：只要是藥水型態的道具在一定時間內目標玩家都無法使用。

強力無所遁形術：

類型：控制

相性：中立

魔力／體力：30 / 0

期間：瞬間

目標：隱身的玩家或擁有隱藏能力的怪物。

範圍：全畫面

材料：無

效果：無所遁形術的強化版，能在全畫面中找出隱身的玩家給予損害並解除隱身。
