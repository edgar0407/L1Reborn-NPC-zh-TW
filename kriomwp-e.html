矮人克里歐姆：

設定倉庫密碼
 1. 點擊「設定/變更密碼」。
2. 點擊「確定」。
3. 輸入新密碼後點擊「確定」。（必須為六位數數字。）
4. 再次輸入新密碼並點擊「確定」以確認新密碼。
5. 倉庫密碼設定完成。

使用倉庫
 1. 點擊「取回道具」。
2. 輸入您設定的六位數密碼後點擊「確定」。
3. 從倉庫取出所需道具。
4. 從血盟倉庫取出道具時，也可使用相同的倉庫密碼。

變更倉庫密碼
 1. 點擊「設定/變更密碼」。
2. 輸入目前密碼後點擊「確定」。
3. 輸入新密碼後點擊「確定」。（必須為六位數數字。）
4. 再次輸入新密碼並點擊「確定」以確認新密碼。
5. 倉庫密碼變更完成。

移除倉庫密碼
 1. 點擊「設定/變更密碼」。
2. 輸入目前密碼後點擊「確定」。
3. 在變更密碼的輸入視窗中不輸入任何內容，直接點擊「確定」。
4. 在確認新密碼的輸入視窗中不輸入任何內容，直接點擊「確定」。
5. 倉庫密碼已移除。

特別注意
 1. 倉庫密碼必須為六位數數字。
2. 您可在 遊戲選項 中設定倉庫密碼。

Kriom the Dwarf:
Setting the Storage House Password: 1.Click Set/Change Password 2.Click OK 3.Enter new password(six-digit) 4.Confirm new password 5.Password set.
Using the Storage House: 1.Click Retrieve Item 2.Enter six-digit password 3.Get items 4.Also works for Blood Pledge storage house.
Changing the Storage House Password: Same as setting but enter current password first.
Removing the Storage House Password: Set/Change Password → enter current → leave new password blank → confirm blank.
Special Note: Must be six-digit number. Can also set at Game Options.
