一般魔法：五級
木乃伊的詛咒：

類型：詛咒、地

相性：邪惡（每次使用增加500點邪惡值）

魔力／體力：80 / 0

期間：32秒

延遲：5秒

目標：1位玩家或非玩家操縱的角色

範圍：1格

材料：魔法寶石x2

效果：有一定的機率麻痺目標，使之無法動彈。麻痺期間的長短依照施法者等級而定。

極道落雷：

類型：攻擊、風

相性：中立

魔力／體力：18 / 0

期間：瞬間

延遲：0.1秒

目標：1位玩家或非玩家操縱的角色

範圍：6格

材料：無

效果：從天空呼叫閃電並在目標上施以嚴重的損害。

高級治癒術：

類型：治癒、水

相性：正義

魔力／體力：20 / 0

期間：瞬間

延遲：0.2秒

目標：1位玩家或非玩家操縱的角色

範圍：全畫面

材料：無

效果：能回復目標大量的體力，它也能使用在不死怪物上並施以損害。

迷魅術：

類型：控制

相性：中立

魔力／體力：30 / 20

期間：1小時

延遲：0.01秒

目標：1個非玩家操縱角色

範圍：全畫面

材料：魔法寶石x1

效果：有一定的機率迷惑目標並使它跟隨施法者的命令。當目標的體力值減少時，這個法術的成功機率會增加；不過某些怪物對這法術是免疫的。

聖潔之光：

類型：生命、水

相性：正義

魔力／體力：20 / 0

期間：瞬間

延遲：0.01秒

目標：施法者本身或1位其他玩家

範圍：全畫面

材料：無

效果：以一定機率解除所有的詛咒。

冰錐：

類型：攻擊、水

相性：中立

魔力／體力：18 / 0

期間：瞬間

延遲：0.1秒

目標：1位玩家或非玩家操縱的角色

範圍：10格

材料：無

效果：在施法者眼前會出現圓形的冷氣，從它出現圓角形的冷氣，再傷害一個攻擊的目標。

魔力奪取：

類型：變化

相性：中立

魔力／體力：1 / 50

期間：瞬間

延遲：0.5秒

目標：1個其他玩家及非玩家操縱角色

範圍：4格

材料：無

效果：將目標的魔力減少，並把減少的部分吸收為自己的魔力。

黑闇之影：

類型：控制

相性：邪惡（每次使用增加10點邪惡值）

魔力／體力：15 / 0

期間：32秒

延遲：0.5秒

目標：1位玩家或非玩家操縱的角色

範圍：3格

材料：魔法寶石x1

效果：將目標的視野變暗，可使用魔法相消術相除魔法效果。
